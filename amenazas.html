Amenazas Cibernéticas y Prevención

Phishing

¿Qué es?

Técnica de ingeniería social que busca obtener información confidencial haciéndose pasar por una entidad confiable.

Características:

  • Emails falsos
  • Sitios web fraudulentos
  • Solicitud de datos personales
  • Urgencia falsa

Cómo prevenirlo:

  • Verificar la dirección del remitente
  • No hacer clic en enlaces sospechosos
  • Verificar URLs antes de ingresar datos
  • Usar autenticación de dos factores
  • Mantener actualizado el navegador
  • Capacitación constante del personal
Ransomware

¿Qué es?

Malware que cifra archivos del sistema y exige un rescate para restaurar el acceso.

Características:

  • Cifrado de archivos
  • Demanda de rescate
  • Propagación rápida
  • Pérdida de datos

Cómo prevenirlo:

  • Realizar copias de seguridad regulares
  • Mantener software actualizado
  • Usar antivirus confiable
  • Evitar descargas de fuentes desconocidas
  • Limitar privilegios de usuario
  • Segmentar la red
Malware

¿Qué es?

Software malicioso diseñado para dañar, infiltrarse o tomar control de sistemas informáticos.

Tipos comunes:

  • Virus
  • Troyanos
  • Spyware
  • Rootkits

Cómo prevenirlo:

  • Instalar antivirus actualizado
  • Evitar software pirata
  • Descargar solo de fuentes oficiales
  • Mantener sistema operativo actualizado
  • Usar firewall
  • Escanear archivos antes de abrirlos
Ataques DDoS

¿Qué es?

Ataque que busca hacer inaccesible un servicio sobrecargándolo con tráfico desde múltiples fuentes.

Características:

  • Tráfico masivo
  • Múltiples fuentes
  • Sobrecarga de servidores
  • Interrupción del servicio

Cómo prevenirlo:

  • Usar servicios de protección DDoS
  • Configurar rate limiting
  • Implementar balanceadores de carga
  • Monitorear tráfico constantemente
  • Tener plan de respuesta a incidentes
  • Usar CDN para distribuir carga

Algunos casos reales de alto impacto

1 / 3
Ataque WannaCry (2017)

Uno de los ataques de ransomware más devastadores de la historia, afectó a más de 300,000 computadoras en 150 países, incluyendo hospitales del Reino Unido, causando la cancelación de miles de citas médicas.

Lección aprendida: La importancia de mantener los sistemas actualizados con parches de seguridad.

Violación de Datos de Equifax (2017)

Los datos personales de 147 millones de personas fueron comprometidos, incluyendo números de seguro social, fechas de nacimiento y direcciones. El ataque se mantuvo sin detectar durante meses.

Lección aprendida: La necesidad de monitoreo continuo y respuesta rápida a incidentes de seguridad.

Ataque de Phishing en Twitter (2020)

Hackers accedieron a cuentas de alto perfil con ingeniería social.

Lección: Capacitar al personal y usar autentificación multifactor.
